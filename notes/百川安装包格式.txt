压缩包结构：以[]开头的代表目录，以*开头代表可选
根
	[]addons 所有附属包
		[]一个附属包
			[]data 该附属包的所有数据
				[] 一个难度，其中的内容将被自动合并至游戏根目录
				*[] 不符合任何难度名称的文件夹，可以存放自定义数据，供安装脚本自由取用
				[]_ 通配难度，无论选定了什么难度，其中的内容将被自动合并至游戏根目录
			[]mods 该附属包的所有模组本体
				[] 一个模组本体，名称随意，里面容纳该模组本体的所有数据，供安装脚本自由取用
			[]difficults 所有难度的脚本。不存在匹配或通配难度的难度时，代表该附属包不支持安装于选定难度
				[] 一个难度，当选定的安装难度匹配该目录名称时，该附属包才能安装
					install.bcis 附属包的安装脚本
					uninstall.bcis 附属包的卸载脚本，附属包被安装后，该脚本将被合并在主卸载脚本的头部
				[]_ 通配难度，无论选定了什么难度，其中的安装脚本都将自动执行，并且内容将被合并在主卸载脚本的头部
			addon.json
	[]data 所有自定义数据
		[] 一个难度，其中的内容将被自动合并至游戏根目录(在通配难度之后)
		*[] 不符合任何难度名称的目录，可以存放自定义数据，供安装脚本自由取用
		[]_ 通配难度，无论选定了什么难度，其中的内容将被自动合并至游戏根目录
	[]difficults 所有难度的脚本
		[] 一个难度
			install.bcis 难度的安装脚本
			uninstall.bcis 难度的卸载脚本
	[]frameworks 前置框架，如BepInEx和QMods，本文件夹的内容将在安装过程中首先被复制到游戏根目录
		[]BepInEx
			[]core
			[]plugins
				...QModsInstaller.dll
			...其他BepInEx组件
		[]QMods
		winhttp.dll
		...其他BepInEx组件
	[]mods 所有模组本体
		[] 一个模组本体，名称随意，里面容纳该模组本体的所有数据，供安装脚本自由取用
	pack.json 安装包元数据，记录一些必要信息

pack.json结构：
{}
	version Num代表当前安装包的版本
	version_name Num代表当前安装包的版本名称
	fork_version Num分支版本，用于在主版本不变的情况进行分支，使得发布分支版本时不影响主版本更新
	difficults Arr容纳所有难度的数据
		{}一个难度
			path Str难度的路径，相对于压缩包的difficults
			name Str难度的名称，用于显示在UI中
	mods Arr容纳所有mod的数据
		{}一个mod
			path Str该mod的路径，相对于压缩包的mods
			name Str该mod的名称，用于在安装脚本中引用
			hash_list []该mod的哈希检查表，用于文件完整性检查
				{}一个该模组中涉及完整性验证的文件的相关信息
					path Str该文件的路径，相对于父级缩进下的path指定的目录
					md5 Str该文件的MD5哈希值
	addons Arr容纳所有附属包数据
		{}一个附属包
			path Str该附属包的路径，相对于压缩包的addons
			name Str该附属包的名称，用于显示在UI中

addon.json结构：
{}
	mods Arr容纳所有附属包内模组的数据
		{}一个模组
			path Str该模组的路径，相对于压缩包的addons
			name Str该模组的名称，用于在安装脚本中引用(由附属包注册的模组也会全局通用，请注意不要与pack.json注册的模组共用相同的名称)
			hash_list []该模组的哈希检查表，用于文件完整性检查
				{}一个该模组中涉及完整性验证的文件的相关信息
					path Str该文件的路径，相对于父级缩进下的path指定的目录
					md5 Str该文件的MD5哈希值
	

安装器操作脚本：
操作脚本采用类似mcfunction的命令行方式进行
命令列表：
q(qmods)
	用于将一个模组文件夹从安装包中复制到"游戏根目录/QMods"中
	语法：
		q <模组引用名称>
	参数：
		模组引用名称：
			填写一个模组在pack.json中注册的引用名称
b(bepinex)
	用于将一个模组文件夹从安装包中复制到"游戏根目录/BepInEx/plugins"中
	语法：
		b <模组引用名称>
	参数：
		模组引用名称：
			填写一个模组在pack.json中注册的引用名称
cf(copyfile)
	用于手动操作文件复制，将单个文件从一处复制到游戏中的一处
	语法：
		cf p <包内路径> <游戏内路径>
		cf g <游戏内路径> <游戏内路径>
	参数：
		包内路径：
			填写一个相对于安装包的文件路径
		游戏内路径：
			填写一个相对于游戏根目录的文件路径
cd(copydir)
	用于手动操作目录复制，将一个目录连同其内部内容从一处复制到游戏中的一处
	语法：
		cd p <包内路径> <游戏内路径>
		cd g <游戏内路径> <游戏内路径>
	参数：
		包内路径：
			填写一个相对于安装包的目录路径
		游戏内路径：
			填写一个相对于游戏根目录的目录路径
d(delete)
	删除游戏中的一个文件或目录
	语法：
		d <游戏内路径>
	参数：
		游戏内路径：
			填写一个相对于游戏根目录的目录路径或文件路径
c(clear)
	清空游戏中一个目录中的所有内容
	语法：
		c <游戏内路径>
	参数：
		游戏内路径：
			填写一个相对于游戏根目录的目录路径
